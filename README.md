## Featured projects

**[sistema-bancario-java](https://github.com/vasqodev/sistema-bancario-java)** — Java desktop application for banking client, account and transfer management. Demonstrates: MVC architecture, Java Swing UI, JDBC + PostgreSQL persistence with PreparedStatement, DAO pattern, polymorphic account hierarchy (abstract Cuenta + CuentaCorriente / CuentaAhorro), BigDecimal arithmetic with banker's rounding, and concurrent transfers using ReentrantLock with id-ordered acquisition to prevent deadlock. Validated with 9 JUnit 5 tests, including 50 concurrent transfers preserving total balance and a deadlock-prevention test under timeout.

**[excel-report-consolidator](https://github.com/vasqodev/excel-report-consolidator)** — Python pipeline that consolidates monthly Excel reports from multiple sources using pandas, openpyxl and xlsxwriter. Includes unit tests with pytest.
